# Create a Gif

# Step 1 Create

- Export animation to series of .PNG files

# Step 2 Import

- Open files in GIMP
  - File > Open as Layers > Select all PNG files
  - Each image imported as a layer
- Reorder Layers
  - Images imported in reverse order
  - Layer > Stack > Reverse Layer Order

# Step 3 Settings

- Optimize
  - Filters > Animation > Optimize (for GIF)

# Step 4 Export

- Export
  - File > Export As > name.gif

# Step 5 Export Settings

- Set the Loop (Loop Forever)
- Adjust delay between frames (milliseconds)
  - If not set, may be too fast or too slow
  - 1000ms = 1 second
- Set Frame Disposal
  - Set to "One frame per layer (replace)" for standard animation
