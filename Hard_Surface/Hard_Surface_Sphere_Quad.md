# Quad Sphere

- Creating a sphere out of Quads



# Create

- Start with a cube
- Add a sub surf modifier, level 3
- Apply the modifier



# Fix Size

- Select all vertices in Edit mode
- Use `SHIFT + ALT + S` > drag all the way to the right



# Fix Corners

- They are the 3 edge vertices, the old corners of the cube (6 of them)
- Select 1
- Go to Select > Select Similar > Amount of Connected edges
- Scale to size: `S 0.9946`
- `CTRL +` to select the vertices around the prior size
- Scale to size `S 0.9983`
