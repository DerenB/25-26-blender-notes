### Alpha Lock (layer edit lock)

- Locks it so you can only edit on the current layer
- Easier to color within the lines
- Tap Layer > Alpha Lock



### Clipping Mask

- Place layer that needs to be clipped on top of limiting layer
- Add clipping mask
- Can't draw outside of other layer



### Color Picker

- Hold finger down until wheel shows



### Duplicate Layer/Shape

- Slide left on the layer > duplicate
- Tap arrow in top left to select
- Drag around to move



### Fill Layer

- Select new color
- Turn on alpha lock for the layer
- Tap on layer > Fill Layer



#### Fill Layer, Partial

- If you want to fill only part of the layer
- Set layer to alpha lock
- Tap wavy "S" icon at top
- Drag select acround what you want filled
- Tap grey circle on selection line to finalize selection
- Layer > Fill Layer like before



### Fill Shape

- Draw outline of what needs to be filled
- Hold on color dot in top right
- Drag onto canvas inside shape


#### Fill Shape, threshhold

- Hold the pen down, threshold shows up at the top
- How to fix how much of the image gets filled



#### Fill Shape, continue

- After filling 1 shape, continue pops up on top of screen
- Tap it.
- Can then tap other shapes to fill them with the same color



### Group Layers

- Select multiple layers with swiping right on layer
- Click Group
- Can rename the group



### Group Layers, Combine together

- Grab the top item and the bottom item in the layer group
- Pull fingers 



### Opacity

- Click 'N' on tha layer
- Drag opacity slider



### Reference Image

- Wrench Icon top Left
- Turn on Reference
- Select Image



### Select All Layers

- Select layer by swiping right on the layer



### Shape, Circle (perfect)

- Draw a circle
- Hold the pen down
- Once it turns into an eclipse, tap 1 finger to make it a perfect circle




### Shape, Eclipse

- Draw the eclipse
- Hold the pen down until the editor shows up



### Undo

- 2 finger tap
