# Geometry Nodes

- Blender Guru's playlist on nodes



# New Node

- Create new Geometry nodes on the Geometry Nodes tab
- This adds a modifier to the object



# Nodes

### Distribute Points on Faces

- Just places the points on the object, not the actual objects



### Instance on Points

- Adds the object instances to the points that were added to the map



### Join Geometry

- Multiple geometry meshes can feed into this
- Send the base mesh from the Group Input into this
- Send the Instance on Points into this as well
