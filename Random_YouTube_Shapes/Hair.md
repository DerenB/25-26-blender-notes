# Hair

- Create an object
- Add particles
  - Properties > Particles
  - Set to "Hair"
  - Enable "Advanced"
  - Modifiers
    - If you have modifiers, check box "Use Modifier Stack
    - Emission > Source



# Settings

- Emission
  - Number
    - Sets the parent particles 
    - Doesn't mean number of hairs
- Hair Dynamics
  - Hair motion
  - Stiffness
    - Under Structure
- Children
  - Can set to Interpolated
  - Multiplies the Emission Number (parent) by the children
  - Has a display vs. render amount
  - Clumping
    - Can set if the base or the ends of the hair clump together
  - Roughness
    - Can set shape, mat, curly, etc.
- Steps
  - Set the Hair Steps
  - Render:
    - Render > Path > B-Spline
    - Set to 4 maybe? (default 3)
  - ViewPort
    - Viewport Display > Strand Steps 5? (default 2)



# Brushing

- Change from Object mode to Particle Edit
- Enable children
  - Only shows parents by default
- Can brush the hair around
