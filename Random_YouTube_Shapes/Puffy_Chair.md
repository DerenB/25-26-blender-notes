# Puffy Chair

- Add Cube
  - Delete front and top face
  - Sub divide once
- Armrests
  - Move the front faces down
  - Move the top edges back
  - Basically set the angle for the armrests
- Add Modifiers
  - Subdiv modifier 2, (ctrl 2)
  - Solidify modifier, set to 0.16
- Apply both modifiers
- Rim Cut
  - Add a Loop Cut around the rim
  - Scale slightly
- Cushion creases
  - Select lines where there should be a crease
  - Bevel > 2 loop cuts
  - DON'T DESELECT THE NEW CUTS
- Object Mode
  - Add another subdiv modifier, apply
- Sculpt mode
  - Top Row > Face Sets > Face Set From Edit Mode Selection
  - Tool
  - Left Column > Cloth Filter
  - Right Side > Properties > Tool > Filter Type
    - Set to Inflate
    - Check "Use Face Sets"
  - Inflate as you want
